# Changelog

## 1.0.0

- Initial release
- Visual Blockly editor with ioBroker-equivalent block categories
- System, Actions, SendTo, Trigger, Timeouts, DateTime, Convert blocks
- Standard Blockly: Logic, Loops, Math, Text, Lists, Colour, Variables, Functions
- Live script execution via Home Assistant WebSocket API
- Real-time entity browser with search
- Live log console with severity levels
- Multi-script management with persistent storage
- Auto-connect via Supervisor API (add-on mode)
- Standalone mode with manual URL/token input
- Astro triggers (sunrise/sunset/dawn/dusk) with offset
- Cron scheduling with full 5-field syntax
- Dark theme optimized for HA interface
